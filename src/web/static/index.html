GemmaBot

Three live demos of Gemma 4 31B driving robots through a Cerebras-hosted inference loop. Each one is a single composite image → one Gemma call → a closed-loop control update.

🦾 localhost:8002

Panda · pick-and-place

Robosuite MuJoCo Panda arm. Gemma sees a stitched birdview+frontside image with proprioception text, returns target XYZ + gripper command, and the OSC controller drives the arm toward it over ~25 physics steps.

live robosuite OSC control ~500ms/call

PushT · hybrid search + Gemma

Gymnasium PushT block-pushing task. A grid-search phase finds the T-block without LLM calls, then Gemma takes over with reward-delta feedback to push it home. Watch the phase transitions live.

live gym-pusht hybrid policy reward feedback

Real robot camera · LeRobot

Real ALOHA cabinet-opening frames from the LeRobot dataset, with a Zone A-F grid overlay. Use the episode/frame controls to step through real robot camera footage and inspect the input Gemma actually sees.
